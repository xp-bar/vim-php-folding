## Vim PHP Folding

#### Installation
Put `folding.vim` under your vim installation directory under a `ftplugin/php/` directory.

For me, for example, that's under `~/.vim/ftplugin/php/folding.vim`, or for nvim `~/.config/nvim/ftplugin/php/folding.vim`.
